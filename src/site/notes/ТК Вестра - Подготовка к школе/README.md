Формат росписи происходит следующим образом:
1) Берётся один вопрос из билета. Как я могу видеть, они не повторяются ни между билетов, ни между школ
2) Вопрос делится на все необходимые составляющие
3) создаются ссылки на простые понятия,
4) и уже они расписываются так, как это следует делать в соответствии с локальным описанием.
