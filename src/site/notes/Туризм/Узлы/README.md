Этот раздел должен быть подвергнут строгой системе описания для наибольшего удобства чтения:

1) Единая картинка с подробным алгоритмом вязания узла
2) Краткое текстовое описание алгоритма навязки узла (Как подпись к картинке и её шагам. Предупреждения об ошибках, потенциальные перехлёсты, и т.д.)
3) Ссылка на видео (Youtube, диск, что угодно), где узел вяжется (В идеале без комментариев. Просто вязка узла, просто, чётко и быстро)
4) Подробное текстовое описание использования узла:
	- Применение
	- Нюансы
	- Похожие узлы
		- Отличительные особенности
	- Ограничения (Ширина и дубовость верёвок, {что-то ещё?})
5) Последовательность картинок, подробно описывающих каждый шаг вязания узла (Для более удобного формата, чтобы можно было вычленить ошибки)
