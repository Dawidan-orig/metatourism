#### Азимут
Оригинальное определение: Это угол между направлением на север и желаемым направлением движения.

Это движение "Напрямик", заключающееся в том, чтобы идти, не меняя направлений, от ориентира к ориентиру.
Проблема: Из-за препятствий, отмеченных карте объектов или просто из-за "Сдувания" в сторону по каким-то причинам, азимут следует применять для перемещения на не очень большие дистанции и от очень характерных ориентиров.

Чтобы нормально азимутовать обязателен компас.

Нейро:
- Перед началом движения азимут определяют по карте с помощью компаса.
- Для удобства маршрут разбивают на этапы с промежуточными ориентирами или точками.
- На местности компас устанавливают горизонтально, фиксируют азимут, выбирают ориентиры в выбранном направлении.
- Двигаются по прямой к ближайшему ориентиру, затем повторяют процесс для следующего этапа.
- При встрече препятствий обходят их по ломаной линии, тщательно фиксируя промежуточные азимуты и расстояния.
- В условиях плохой видимости используют метод створа — последовательное движение между созданными ориентирными точками.
- Для уменьшения ошибок регулярно сверяются с компасом и при необходимости корректируют курс.

#### Счисление пути на местности
- Дистанцию измеряют шагами (2 шага = примерно 1.5 м для среднего человека), либо подсчётом времени относительно скорости движения.
- В сложной местности путь делят на отрезки с записями пройденного расстояния и изменений направления.
- Ведут схематический рисунок пути с указанием направлений (азимутов) и длины отрезков.
- Для возврата по маршруту используют обратные азимуты.
- При наличии ориентиров сверка по ним помогает не сбиваться с маршрута.
- При движении по незнакомой местности важно отмечать пройденные точки и делать пометки для безопасности.
